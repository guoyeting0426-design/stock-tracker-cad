# Tracker Online (GitHub Pages)

## First publish (web UI)
1. Create a new GitHub repository (public), e.g. `stock-tracker-cad`.
2. Upload all files from this folder to repo root.
3. In GitHub repo: Settings -> Pages -> Build and deployment -> Source: `GitHub Actions`.
4. Wait for Action "Deploy GitHub Pages" to finish.
5. Your link will be:
   `https://<your-github-username>.github.io/<repo-name>/`

## Update later
1. Edit `index.html` in GitHub web editor.
2. Commit changes to `main`.
3. Wait ~1 minute; site auto-updates.
